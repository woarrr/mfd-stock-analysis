# mfd-stock-analysis

работа по дицсиплине "Анализ Данных"

Проект посвящен статистическому анализу и первичной обработке финансовых данных трех тикеров. В ходе работы проводится очистка данных от аномалий, расчет описательных статистик и проверка гипотез о нормальности распределения.

**Внимание:** В соответствии с правилами использования данных Московской Биржи, исходные файлы не включены в репозиторий. Для запуска проекта необходимо самостоятельно загрузить котировки для тикеров Транснефть-ап, Красный Октябрь-ао и ИРКУТ-3 и поместить их в корневую директорию под соответствующими именами.

**Тикер 1:** Транснф ап  
**Тикер 2:** КрасОкт-ао  
**Тикер 3:** ИРКУТ-3  

Начало временного интервала:         2008-12-22
Окончание временного интервала:   2022-07-19

#### этапы работы
1. Загрузка и предобработка данных 
- Чтение данных из CSV-файлов 
- Удаление лишних столбцов 
- Расчет логарифмической доходности для каждой компании 
- Удаление строк с нулевой доходностью и проверка синхронизации временных рядов 

2. Первичный анализ
- Расчет объема, выборочного среднего и несмещенной оценки стандартного отклонения 
- Расчет корелляции 
- Визуализация данных в виде диаграмм рассеивания 
- Построение диаграммы Парето 

3. Очистка от выбросов и анализ
- Построение boxplot
- Использование метода межквартального размаха для выявления и удаление выбросов логдоходности 
- Расчет объема выборок, размаха, среднего, медианы, СКО
- Расчет эскцесса и ассиметрии 
- Построение 95% доверительного интервала для математического ожидания
- Построения 95% доверительного интервала для дисперсии 
- Построение гистограмм интервальных частоты 

#### технологический стек
- python 3.14.2
- Pandas
- NumPy
- SciPy.stats 
- Matplotlib
- Seaborn 

#### Запуск

1. Добавьте файлы с данными в корневую папку проекта 
2. Установитье зависимости:
```pip install pandas numpy scipy matplotlib seaborn```
3. Запустите Jupiter Notebook stock-analysis.ipynb
